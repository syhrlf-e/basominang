# Aset Branding BasoMinang

- `basominang.ico` — icon utama Windows untuk installer dan shortcut; berisi ukuran 16, 32, dan 48 px.
- `basominang.png` — artwork sumber resolusi tinggi.
- `basominang-32.ico` — varian icon 32 px awal, disimpan sebagai arsip.
- `web/` — aset favicon dan PWA hasil generator icon, dipisahkan dari root project agar tidak mengganggu source utama.

Extension VS Code memakai salinan `512 × 512` pada `vscode-extension/assets/basominang.png` agar paket VSIX tetap ringan.
